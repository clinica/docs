# Ação: ver aniversariantes

## Cenário real
Você quer fazer ações simples de relacionamento: mandar uma mensagem de aniversário, ligar para pacientes importantes ou organizar uma campanha do mês.

## Objetivo
Ver uma lista de pacientes aniversariantes para orientar contato e relacionamento.

## Quando usar
- Quando a clínica quer enviar felicitações ou campanhas de relacionamento.
- Quando deseja organizar ligações/mensagens para aniversariantes do mês.

## Quando NÃO usar
- Se a tela estiver vazia/sem dados no momento: use alternativas (lista manual, exportação) até o recurso ficar completo.

## Passo a passo detalhado
1. Acesse **Pacientes**.
2. Abra o menu de opções (ícone de “mais opções”).
3. Clique em **Aniversariantes**.

## Campos envolvidos
Pode variar conforme evolução do recurso. No momento, a tela pode estar em estado inicial.

## Regras do trabalho (como usar no dia a dia)
- Se a lista estiver disponível, defina uma rotina (ex.: toda segunda-feira ver aniversariantes da semana).

## Possíveis erros e como resolver
- **Tela simples/sem conteúdo**:
  - Indica que a funcionalidade ainda não foi finalizada.
  - Solução: acompanhe releases/atualizações do sistema.

## Resultado esperado
Quando completo, deve listar pacientes aniversariantes com filtros e ações de contato.

## Observações importantes
- Se você precisa do recurso imediatamente, a alternativa é exportar/filtrar manualmente a base de pacientes.

## Apêndice técnico (para suporte)
- Esta rota pode existir antes da UI final estar implementada.
