# Ajustes: corpo clínico

## Objetivo
Organizar a equipe no sistema: quem é profissional (atende) e quem é usuário (apoio/administrativo), com acessos alinhados para cada função.

## Para quem é
- **Gestor**.

## Ações disponíveis
- [`acao-gerenciar-profissionais-e-usuarios.md`](acao-gerenciar-profissionais-e-usuarios.md)

## Como essa área entra no seu dia
- Entrou alguém novo na equipe → cadastrar aqui.
- Alguém mudou de função → ajustar acesso para não ficar sem permissão (ou com permissão demais).
