# Ação: visualizar dados arquivados

## Descrição da ação
Acessar seções específicas para ver itens arquivados e, quando disponível, desarquivar para voltar ao uso normal.

## Quando usar
- Quando um convênio/serviço/fornecedor foi arquivado por engano.
- Quando precisa recuperar um cadastro antigo para uso novamente.

## Passo a passo detalhado
1. Acesse **Ajustes → Dados Arquivados**.
2. Escolha a categoria desejada:
   - Convênios arquivados
   - Serviços arquivados
   - Fornecedores arquivados
3. Na tela específica, execute a ação de desarquivar conforme a UI disponível.

## Campos envolvidos
Depende da categoria escolhida e da tela de listagem/recuperação.

## Regras de negócio
- A página principal funciona como “hub” com botões para cada tipo de dado.

## Possíveis erros e como resolver
- **Lista vazia**:
  - Pode indicar que não existem itens arquivados.

## Resultado esperado
Você consegue localizar itens arquivados e restaurá-los para uso no sistema.
